# Meu App

Essa é uma aplicação web desenvolvida com Next.js e integração com Supabase, com foco em front-end e manipulação de dados.

## Funcionalidades
- Cadastro de usuários;
- Integração com banco de dados (supabase);
- Interface responsiva;
- Manipulação dinâmica de dados

## Tecnologias/Linguagens

- Next.js
- TypeScript
- JavaScript
- Supabase
- HTML & CSS
